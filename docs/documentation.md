## 🎨 কাস্টমাইজার প্যানেল ব্যবহার করুন

### 1️⃣ রঙের থিম পরিবর্তন করুন

**উপলব্ধ থিম:**
- 🔵 **ডিফল্ট** - নীল রঙের সুন্দর থিম
- 🌙 **ডার্ক** - অন্ধকার মোডের জন্য আদর্শ
- 🌊 **সমুদ্র** - নীল এবং সাইয়ান রঙের সংমিশ্রণ
- 🌅 **সূর্যাস্ত** - কমলা এবং লাল টোন
- 🌳 **বন** - সবুজ রঙের প্রাকৃতিক থিম

**পরিবর্তনের পদ্ধতি:**
1. কাস্টমাইজার প্যানেল খুলুন
2. "রঙের থিম" সেকশনে যান
3. পছন্দের থিম নির্বাচন করুন
4. পরিবর্তন স্বয়ংক্রিয়ভাবে প্রয়োগ হয়

### 2️⃣ সাইট তথ্য সম্পাদনা করুন

**সাইটের নাম এবং ট্যাগলাইন পরিবর্তন করুন:**

```
সাইট তথ্য বিভাগ:
├── সাইটের নাম: আপনার সাইটের নাম লিখুন
├── ট্যাগলাইন: একটি সংক্ষিপ্ত বর্ণনা লিখুন
└── আপডেট করুন বোতামে ক্লিক করুন
```

### 3️⃣ বিজ্ঞাপন সেটিংস কনফিগার করুন

**Google AdSense কোড যুক্ত করুন:**

1. আপনার Google AdSense অ্যাকাউন্টে লগইন করুন
2. বিজ্ঞাপন কোড কপি করুন
3. কাস্টমাইজার প্যানেলে আটকান
4. "বিজ্ঞাপন কোড সংরক্ষণ করুন" বোতামে ক্লিক করুন

**বিজ্ঞাপন অবস্থান:**
- 📍 শীর্ষ - হেডারের নিচে
- 📍 মধ্য - টুলস গ্রিডের নিচে
- 📍 নিচ - ফুটারের উপরে

### 4️⃣ সোশ্যাল মিডিয়া লিংক যুক্ত করুন

**লিংক যুক্ত করার পদ্ধতি:**

```
সোশ্যাল মিডিয়া সেকশনে:
├── Facebook URL: https://facebook.com/yourpage
├── Twitter URL: https://twitter.com/yourhandle
├── Instagram URL: https://instagram.com/yourusername
├── YouTube URL: https://youtube.com/yourchannel
├── LinkedIn URL: https://linkedin.com/in/yourprofile
└── সংরক্ষণ করুন
```

### 5️⃣ টুলস ম্যানেজ করুন

**নতুন টুল যুক্ত করুন:**

1. টুলস ম্যানেজমেন্ট সেকশনে যান
2. টুলের নাম লিখুন (উদাহরণ: "PDF to Word")
3. টুলের URL লিখুন (যেখানে টুলটি অবস্থিত)
4. ইমোজি বা আইকন লিখুন (উদাহরণ: 📄)
5. "টুল যোগ করুন" বোতামে ক্লিক করুন

**টুল সরানোর পদ্ধতি:**
- যুক্ত করা টুলের পাশে "অপসারণ" বোতাম আছে
- ক্লিক করুন এবং নিশ্চিতকরণ দিন

**ডিফল্ট টুলস:**
- 📊 Word Counter - শব্দ গণনা
- 🔊 Text to Speech - পাঠ্য থেকে বাণী
- 📱 QR Code Generator - QR কোড তৈরি
- {} JSON Formatter - JSON ফরম্যাট করুন
- 🔐 Password Generator - পাসওয়ার্ড তৈরি করুন
- 🔗 MD5 Hash - MD5 হ্যাশ তৈরি করুন

### 6️⃣ লেআউট অপশন

**উপলব্ধ অপশন:**
- ☑️ সাইডবার দেখান
- ☑️ ফুটার দেখান
- ☑️ বিজ্ঞাপন দেখান

### 7️⃣ ফন্ট আকার সমন্বয় করুন

**ফন্ট সাইজ রেঞ্জ:** 12px থেকে 24px

1. স্লাইডার নড়ান
2. প্রিভিউ তাৎক্ষণিক আপডেট হয়

## 📊 থিম ম্যানেজমেন্ট

### সেটিংস এক্সপোর্ট করুন

```javascript
// কাস্টমাইজার প্যানেলে "সেটিংস এক্সপোর্ট করুন" ক্লিক করুন
// একটি JSON ফাইল ডাউনলোড হবে
// এটি ব্যাকআপ হিসাবে রাখুন
```

### সেটিংস রিসেট করুন

```
রিসেট অপশনস সেকশনে:
├── সব সেটিংস রিসেট করুন - সবকিছু ডিফল্টে ফিরিয়ে দেয়
└── তথ্য: এটি ২৪ ঘন্টার মধ্যে অপরিবর্তনীয়
```

## 💻 প্রযুক্তিগত বিবরণ

### ফাইল স্ট্রাকচার

```
blogger-theme/
├── index.html                      # মূল HTML ফাইল
├── theme/
│   ├── css/
│   │   ├── main.css               # প্রধান স্টাইল
│   │   ├── responsive.css         # রেসপন্সিভ ডিজাইন
│   │   └── animations.css         # অ্যানিমেশন
│   ├── js/
│   │   ├── storage.js             # লোকাল স্টোরেজ
│   │   ├── tools.js               # টুলস ম্যানেজার
│   │   ├── customizer.js          # কাস্টমাইজার
│   │   └── app.js                 # মেইন অ্যাপ্লিকেশন
│   └── assets/
│       └── images/                # ছবি ফোল্ডার
└── docs/
    └── documentation.md           # এই ফাইল
```

### প্রযুক্তি ব্যবহৃত

- **HTML5** - সিমান্টিক মার্কআপ
- **CSS3** - আধুনিক স্টাইলিং এবং অ্যানিমেশন
- **JavaScript (ES6+)** - ইন্টারেক্টিভ ফিচার
- **LocalStorage** - ডেটা পার্সিস্টেন্স
- **Intersection Observer** - পারফরম্যান্স অপটিমাইজেশন

### পারফরম্যান্স মেট্রিক্স

- **পেজ লোড সময়:** < 2 সেকেন্ড
- **ফাইল সাইজ:** < 500 KB (সম্পূর্ণ)
- **মোবাইল স্কোর:** 95+/100
- **ডেস্কটপ স্কোর:** 98+/100
